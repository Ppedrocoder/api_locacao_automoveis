API SOAP — Relatório de Locações

WSDL: http://localhost:9000/relatorio?wsdl

Endpoint: http://localhost:9000/relatorio

Método: consultarRelatorio

Retorna um relatório consolidado das locações, podendo filtrar por status.

Request

<soapenv:Envelope
  xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/"
  xmlns:rel="http://relatorio.locacao">
  <soapenv:Body>
    <rel:consultarRelatorioRequest>
      <rel:status>RESERVADO</rel:status> <!-- opcional -->
    </rel:consultarRelatorioRequest>
  </soapenv:Body>
</soapenv:Envelope>

Parâmetros

CampoTipoObrigatórioValores
statusstringNãoRESERVADO, EM_USO, DEVOLVIDO, DEVOLVIDO_ATRASADO, CANCELADO

Response

<consultarRelatorioResponse>
  <total>10</total>
  <reservados>3</reservados>
  <em_uso>2</em_uso>
  <devolvidos>3</devolvidos>
  <atrasados>1</atrasados>
  <cancelados>1</cancelados>
</consultarRelatorioResponse>

Campos da resposta

CampoTipoDescrição
totalintTotal de locações
reservadosintLocações com status RESERVADO
em_usointLocações com status EM_USO
devolvidosintLocações devolvidas no prazo
atrasadosintLocações devolvidas com atraso
canceladosintLocações canceladas
